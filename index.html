About Time Reporter

Time Reporter is a meeting timer and reporting assistant built specifically for Toastmasters clubs. It helps time speeches, store meeting records, and generate visual reports for club officers and members.

Core features
  • Multiple timing rules with color cues (green / yellow / red)
  • Meeting records & speech data storage
  • Visual report generation (PDF / image export)
  • Custom rules with iCloud sync

User Guide

  1. Timer Tab: Select meeting, speaker, and rule → Start timer → Follow background color changes.
  2. Meetings Tab: Browse historical meetings, tap to expand speaker details.
  3. Reports Tab: Generate charts and export PDF/image for sharing.
  4. Settings Tab: Manage built-in rules or create custom thresholds and alerts.

FAQ

Do I need an internet connection to use the app?
No. Timing and data storage work offline. iCloud is only used for syncing rules across devices.
How can I export reports?
Open the Reports tab → Select a meeting → Tap "Generate Report" → Save as PDF or image → Share.
How is my data protected?
Speaker names are stored locally and anonymized. Meeting data is encrypted on device. You can delete all data from the app anytime.
Which devices are supported?
iPhone and iPad running iOS 15 or later.
